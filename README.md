# Repositiory for the official website of Karl Uwe Martin

Powered by: 
https://nextjs.org/ 
https://mui.com/material-ui/

Official website:
https://karluwemartin.de

## Run
```bash
npm install
npm run dev
```
Then open http://localhost:3000

## Notes
- Uses Next.js App Router (`/app`).
- MUI uses Emotion as the styling engine.
